Package: grails.datastore.test

[Groovy] Class DatastoreUnitTestMixin

    • grails.datastore.test.DatastoreUnitTestMixin

    • A Groovy mixin used for testing datastore interactions. Test cases should include the mixin using the Groovy

      Mixin:
      transformation:

      @Mixin(DatastoreUnitTestMixin) class MyTests {}

      The DatastoreUnitTestMixin.connect method should be called in the test cases setUp() method and the DatastoreUnitTestMixin.disconnect method on the tearDown() method.

      Graeme Rocher
